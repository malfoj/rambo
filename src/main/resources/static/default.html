eRambo.pl by Malfoj

How it works:

Requests shall be sent over (post, get, put, patch, delete, head, options) https://erambo.pl/hook/#CUSTOMNAME# so for example https://erambo.pl/hook/rambo
To see your requests on specific hook go via browser to https://erambo.pl/ui/#CUSTOMNAME# so for example https://erambo.pl/ui/rambo
To change a hook response use patch method over url: PATCH https://erambo.pl/response/#CUSTOMNAME# so for example https://erambo.pl/response/rambo
To get raw entries u can call endpoint of GET https://erambo.pl/raw/#CUSTOMNAME# so for example https://erambo.pl/raw/rambo
To get your current response on a hook u can call endpoint of GET https://erambo.pl/response/#CUSTOMNAME# so for example https://erambo.pl/raw/rambo
